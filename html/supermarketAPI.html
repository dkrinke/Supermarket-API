SupermarketAPI

Supported Endpoints

  • GET
    • Health check: http://api.krinkes.com/healthcheck
      • Returns: "Still Alive!"
    • Get all produce: http://api.krinkes.com/api/v1/produce
      • Returns: List of Produce
    • Get produce by code: http://api.krinkes.com/api/v1/produce/A12T-4GH7-QPL9-3N4M
      • Returns: A single Produce that matches the provided code
  • POST
    • Add produce: http://api.krinkes.com/api/v1/produce
      • Payload:
        {
            "Name": "Green Apples",
            "Code": "123A-123A-123A-124A",
            "Price": "$1.11"
        }
      • Returns: Success or Failure with messages containing the reason
    • Reset produce to default: http://api.krinkes.com/api/v1/produce/reset (Demo purposes)
      • Payload: None
      • Returns: 204 No Content
  • DELETE
    • Remove produce: http://api.krinkes.com/api/v1/produce/123A-123A-123A-124A
      • Payload: None
      • Returns: Success or Failure with messages containing the reason
